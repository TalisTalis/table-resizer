# TableResizer – библиотека для изменения ширины столбцов таблиц

**TableResizer** — это лёгкая JavaScript-библиотека, которая позволяет пользователям изменять ширину столбцов таблиц простым перетаскиванием правой границы заголовка. Все изменения автоматически сохраняются в `localStorage` и восстанавливаются при следующей загрузке страницы. Библиотека также умеет подбирать оптимальную ширину столбцов на основе содержимого и типа данных, а также отображает всплывающие подсказки с полным текстом, если он не помещается в ячейку.

## Возможности

- **Интерактивное изменение ширины** – перетаскивание специального маркера (`resize-handle`) в заголовках столбцов.
- **Сохранение состояния** – ширина каждого столбца сохраняется в `localStorage` и восстанавливается после перезагрузки.
- **Автоподбор ширины** – двойной клик по маркеру автоматически подгоняет ширину столбца под самое длинное содержимое.
- **Умные начальные ширины** – если сохранённых данных нет, библиотека определяет ширину на основе атрибута `data-type`, текста заголовка или CSS-классов (поддерживаются типы: `date`, `phone`, `email`, `fullname`, `action`, `checkbox` и многие другие).
- **Отключение столбцов** – можно запретить изменение ширины для конкретных столбцов (например, для колонки с действиями или чекбоксами).
- **Подсказки при обрезании** – при наведении на заголовок, если текст не помещается, отображается всплывающая подсказка с полным содержимым.
- **Гибкая настройка** – множество опций: минимальная ширина, задержка сохранения, единицы измерения (px, ch, em, rem), собственные типы данных и т.д.

## Установка

Скачайте файл `table-resizer.js` (или `table-resizer.min.js`) и подключите его к вашей HTML-странице:

```html
<script src="path/to/table-resizer.js"></script>
```

## Использование

1. Добавьте таблице уникальный `id`.
2. Убедитесь, что у каждого заголовка (`<th>`) есть атрибут `data-column` с уникальным идентификатором столбца (например, `data-column="name"`). Это необходимо для сохранения ширин даже при перестановке столбцов.
3. Вызовите инициализацию:

```javascript
// Инициализация таблицы с ID "myTable"
window.tableResizer.init('myTable');
```

## Расширенная настройка

```javascript
window.tableResizer.init('myTable', {
    minWidth: 80,                       // минимальная ширина столбца в пикселях
    defaultWidth: 200,                  // ширина по умолчанию (если не определена автоматически)
    autoResizeOnDblClick: true,         // автоподбор по двойному клику
    saveDelay: 500,                     // задержка перед сохранением (мс)
    disabledColumns: ['actions', 'id'], // столбцы, для которых ресайз отключён
    dataColumnAttribute: 'data-column', // атрибут с ключом столбца
    dataTypeAttribute: 'data-type',     // атрибут с типом данных (для умной ширины)
    initialColumnWidths: {              // начальные ширины (при первом запуске)
        name: '20ch',
        date: '12ch',
        status: '10ch'
    },
    dataTypeWidths: {                   // переопределение ширин для типов данных
        'date': '14ch',
        'phone': '18ch'
    }
});
```

## API

### Методы

| Метод | Описание |
| :--- | :--- |
| `init(tableId, options)` | Инициализирует таблицу с указанным ID. Возвращает `true` при успехе. |
| `reset(tableId)` | Сбрасывает ширины столбцов для таблицы (удаляет сохранённые данные и применяет ширины по умолчанию). |
| `destroy(tableId)` | Полностью удаляет функциональность ресайза для таблицы (убирает обработчики, классы и стили). |
| `distributeColumns(tableId)` | Равномерно распределяет ширину между активными столбцами (полезно при изменении размера окна). |

### Опции (конфигурация)

| Опция | Тип | По умолчанию | Описание |
| :--- | :--- | :--- | :--- |
| `storageKey` | `string` | `'tableColumnsWidths_<tableId>'` | Ключ в `localStorage` для хранения ширин. |
| `minWidth` | `number` | `50` | Минимальная ширина столбца в пикселях. |
| `autoResizeOnDblClick` | `boolean` | `true` | Включить автоподбор ширины по двойному клику. |
| `saveDelay` | `number` | `300` | Задержка перед сохранением (мс) для уменьшения частоты записи. |
| `defaultWidth` | `number` | `300` | Ширина по умолчанию (если не удалось определить автоматически). |
| `minColumnsForDefaultWidth` | `number` | `4` | Если активных столбцов больше этого числа, применяется `defaultWidth`, иначе ширина вычисляется как `ширина_экрана / количество_столбцов`. |
| `disabledColumns` | `array` | `[]` | Массив ключей столбцов (из `data-column`), для которых ресайз отключён. |
| `dataColumnAttribute` | `string` | `'data-column'` | Имя атрибута, содержащего уникальный ключ столбца. |
| `dataTypeAttribute` | `string` | `'data-type'` | Имя атрибута, указывающего тип данных (для умного подбора ширины). |
| `initialColumnWidths` | `object` | `{}` | Объект `{ ключ: ширина }` – начальные ширины при первом запуске. |
| `dataTypeWidths` | `object` | *(встроенные)* | Объект соответствия типов данных и их ширины (можно переопределить). |

## Поддерживаемые типы данных (умная ширина)

Библиотека автоматически определяет ширину для следующих типов (задаются через `data-type` или угадываются по заголовку):

| Тип | Пример ширины | Описание |
| :--- | :--- | :--- |
| `date` | `12ch` | Дата (ДД.ММ.ГГГГ) |
| `short-date` | `10ch` | Короткая дата |
| `datetime` | `18ch` | Дата и время |
| `time` | `8ch` | Время |
| `age` | `7ch` | Возраст |
| `gender` | `7ch` | Пол |
| `policy` | `20ch` | Номер полиса (ЕНП) |
| `phone` | `15ch` | Телефон |
| `email` | `25ch` | Email |
| `inn` | `12ch` | ИНН |
| `snils` | `14ch` | СНИЛС |
| `boolean` | `8ch` | Да/Нет |
| `number` | `10ch` | Число |
| `currency` | `12ch` | Деньги |
| `percent` | `8ch` | Процент |
| `status` | `10ch` | Статус |
| `id` | `8ch` | ID |
| `name` | `20ch` | Имя |
| `surname` | `25ch` | Фамилия |
| `patronymic` | `25ch` | Отчество |
| `fullname` | `40ch` | ФИО |
| `address` | `40ch` | Адрес |
| `city` | `20ch` | Город |
| `code` | `10ch` | Код |
| `action` | `40px` | Действия (иконки/кнопки) |
| `checkbox` | `30px` | Чекбокс |
| `hospital_type` | `80px` | Тип стационара |

💡 Вы можете дополнить или переопределить эти значения через опцию dataTypeWidths.

Поддерживаются все современные браузеры (Chrome, Firefox, Safari, Edge), а также Internet Explorer 11 (при использовании полифилов для Map и Array.from).
